Keyboard and mouse controls:

Middle Mouse = Open/Exit edit mode
LMB = Place mesh or Deathsphere

Q and E = Cycle through meshes
RMB = Summon Deathsphere

Mouse up/down + Shift = Change uniform scale
Mouse up/down + Shift + LMB = Change height scaling
Mouse up/down + Shift + RMB = Change depth
Mouse up/down + Shift + LMB + RMB = Change width

Mouse up/down + Left Control + LMB = Change height location
Mouse up/down + Left Control + RMB = Change sideways location
Mouse up/down + Left Control = Change distance from player

Mouse up/down + Left Alt + LMB = Change pitch rotation
Mouse up/down + Left Alt + RMB = Change roll rotation
Mouse up/down + Left Alt + LMB + RMB = Change yaw rotation



Gamepad controls:

Back button = Open/Exit edit mode
Right thumbstick press = Place mesh or Deathsphere

Left and right D-pad = Cycle through meshes
Face button right = Summon Deathsphere

Right thumbstick up/down + Face button left = Change uniform scale
Right thumbstick up/down + Face button left + Left trigger = Change height scaling
Right thumbstick up/down + Face button left + Right trigger = Change depth
Right thumbstick up/down + Face button left + Both triggers = Change width

Right thumbstick up/down + Left shoulder + Left trigger = Change height location
Right thumbstick up/down + Left shoulder + Right trigger = Change sideways location
Right thumbstick up/down + Left shoulder = Change distance from player

Right thumbstick up/down + Right shoulder + Left trigger = Change pitch rotation
Right thumbstick up/down + Right shoulder + Right trigger = Change roll rotation
Right thumbstick up/down + Right shoulder + Both triggers = Change yaw rotation
